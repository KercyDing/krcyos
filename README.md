# 🌱 KrcyOS

A toy RISC-V kernel built with Zig for OS exploration.

## Motivation

Want to understand computer architecture and operating systems.

## Prerequisites

- **Zig** (>=0.16.0)
- **QEMU** (`qemu-system-riscv64`)
- **[Only](https://github.com/KercyDing/only)** (task runner)

## Getting Started

```bash
git clone https://codeberg.org/Kercy/krcyos.git
cd krcyos
# ...
```
